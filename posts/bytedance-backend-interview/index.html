8/18号10.到上海面的头条后端开发岗，这里写一下面经，回报社会，头条很迅速，3面全部一天解决.

offer get

一面

面试官非常和蔼，很好说话。他说面试流程是，1.项目，2.基础(OS,network,组成原理),3.算法，问我ok吗. 当然ok了啊~

我先说了下项目，因为本科生还没干过啥，就说了一下和老师做的科研的项目，是一个关于算法的，感觉没加分…

然后问基础:

  1. 假设一个大型游戏有10+G，内存仅有2G，怎么让它跑起来？

我说是虚存吗，我直接说虚存的设计吧，msg:嗯,ok~,

wo: balabala ~… 说了虚存，从一条CPU打过来的指令开始解释，感觉还不错，

msg:紧接着又问了页表是每个进程都有的还是仅有一个。二级页表的作用，进程的逻辑空间大小看什么。反正问道你答不出来，或者他觉得ok了为止

网络

解释一下一个数据包从一个子网到另一个子网的过程，MAC地址，IP怎么转化的，说清楚

算法题，写代码

  1. 怎么判断一个有向无环图有没有环(topo)
  2. 一颗二叉树的直径，我跟他说两次bfs就好了，他不信，还让我证明了一下…啊艰难的证明

还问了一下java基础，让我解释hashmap

msg:ok,有什么想问我的？

wo:可以给出一个评价吗？msg说还ok，说头条比较看重沟通，说我这部分干的还行。

3分钟后2面通知就来了

二面

1.看简历才知道我是本科生，说他从来没面过本科生。想让我介绍一下。

说一个解决的ACM最难的题目

wo: what???都退役一年了。说了个上次newcoder上做的。解释了好大半天

语言

  1. 说一下c++和java的区别 这里扯了一堆，

  2. 从c++切到java要注意啥

wo： what？？？？没懂，能说详细点吗

msg：student1=student2,java和c++有啥区别

wo: 终于听懂了

  1. 多太了解？？
class base{print}
class supper exend base{。。。@override print()}
base b = new supper();
b.printf();

输出哪个的.

wo: supper

msg:为什么？

不太清楚

msg: 你可以猜一下

我说，因为b指向的内存是supper，然后…

msg： 那你认为这是在编译器决定的还是运行时，根据你答的

w: runtime

ok算法来了

算法:

  1. 内存仅有1kw，有1亿的int在磁盘上 O(n) 找第k大。 我说如果内存没限制我倒是有 O(n)的做法。他说不行，给个提示，值是均匀的1亿~ 2亿,最终提示后我解决了
  2. 二叉树后续遍历，非递归
  3. 一个笔试题，(貌似他们不知道有笔试…)

一个开放问题，他说超纲了，应该是想拒绝我了(55555555555~)

数据库存在不同的服务器上，为了让不同的服务器负载均衡，现在加一台服务器怎样才能让数据搬迁最少。我答了用hash，再hash借鉴hashmap的策略。

msg: 有什么问题要问我吗？ wo: 评价一下我…

3面3分钟后到，2面说3面是他boss.

三面

3面问的很快，似乎想知道你知识的上界在哪里。

介绍

java JVM，JNI，Spring

wo: 不知道…

msg: 语言基础是吧

w: en~

算法题

第k大，写代码，我就写了快排的思想

网络

  1. 3次握手，4次挥手

    我说我3次握手后，他问为什么。我还没答呢，

    他说不知道可以说不知道，不要乱答。

  2. 紧接着他就问了 滑动窗口后拥塞。也是问为什么，我说拥塞是为了，TCP公平。msg说不是。。。哎这里每次都错

  3. 问了一个比较难的题

问我搜索系统，几千万个网页，查询怎么做，

哎，答的不好。后来就吃午饭了 吃午饭时和3面msg一起吃的，他问了我一些其他问题，感觉有点套路
