MIT 6.828 环境配置，踩坑指南

2018年11月27日
| Operating System
| 阅读
文章目录
  • git clone err

我的环境 :

  1. win10, git 2.19
  2. Ubuntu 18.04 git 2.17
  3. jos.git year : 2018

git clone err

  1. SSL err

直接 git clone git clone https://pdos.csail.mit.edu/6.828/2018/jos.git lab

可能会出现

fatal: unable to access 'https://pdos.csail.mit.edu/6.828/2018/jos.git/': OpenSSL SSL_connect: SSL_ERROR_SYSCALL in connection to pdos.csail.mit.edu:443

解决方法

添加两行就行了

export GIT_CURL_VERBOSE=1
export GIT_TRACE_PACKET=2

注意 这个方法在win10 下能成功安装，但是 Ubuntu 下仅能 clone 2017 年的版本
