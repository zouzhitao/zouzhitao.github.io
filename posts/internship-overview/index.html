转眼来这里实习已经有1月了，干了一些活，也有了一些感悟，准备简单小结一下

life

这边的生活还是很轻松的，mentor手下就我和一个北大的研究生 两个intern, 我两做的东西不一样，虽然大方向是一样的，做的都是 interpretable ML 方面的东西(PS:其实这就是换了一个名字，具体来说就是用weak的model，能看懂数学形态或者数学表达式的model去解释data)本质上就是一种预测能力弱但是解释能力强的model。最近我做的这个被选到 MS TechFest 上了，3月份会被带到总部展览。虽然我不能去，但还是很fantastic。

这边有安排宾馆，也可以自己找房子，我直接住的宾馆，和我同住的是一位 USTC的phd，他来msra已经两次了。大佬特别拼周末啊，晚上啊都去公司。可能是因为他做的东西更偏向发表论文吧，我做的偏工程一些，就是将一些论文中的知识变成model，变成实际的产品，这也更符合我自己的期望吧。顺便说一下，我个人认为亚洲研究院这边大概有3波人，一种是做前沿研究的，真正的scientist，比如做量子计算，理论之类的真正特别硬核的强者.另外一种就是我这种了，做的是将论文中的东西，将知识变成有前景的东西，或者更直接直接应用到微软的产品中。所以 Microsoft research 每年都会举办TechFest, 就是研究院的人向所有微软的产品部门展示自己的成果。我做的这个东西居然被选中了，不得不说也是一种缘分吧。还有一波人可能介于两者之间吧，做语音，图形图像，系统，方面的应用研究，不过这边的研究方向还是很多的，基本各种方向都有，不过都比较贴近微软的产品。和微软产品无关的应用研究基本就没有，比如说无人驾驶，这边就没人做这个。

这边比较舒服的一点可能是没有传统互联网公司的PM这种东西来给你提需求，整个东西就是你觉得你研究的东西能拿来干什么自己把它实现出来就行了。完全是由自己从头打造一个系统，这种感觉太棒了什么都由你自己决定，你想怎么设计就怎么设计。本来我做的这个东西以前是以为pku的本科生在做后来他走了，然后我就接着他的工作做下去了，令我感到非常坑的是他走的时候忘记code review 了导致一部分轮子也没有实现，搞得我得全部实现所有的东西，所以就变成整个东西都是我一个人实现的了。

这边每周都有一些talk没事的时候可以去听，有些是专门请微软的高级研究员讲的怎么做研究的讲座，有些外面的或者微软内部人的一些学术报告。来这里听了4次，两次学术talk，两次研究talk，一次是topo量子计算的，没听太懂，就听懂了它最后说的，现在媒体鼓吹的已经实现了100+ bit量子计算机都是假的，那是量子bit，不是实际的逻辑bit，从量子bit到逻辑bit还有很大的距离，就像从牛顿第二定律到登上月球那样遥远。

接下来是我做的一些事情

my work

我这边是先用一些简单的语言，开发方便的语言，py这种快速实现算法，验证trick的可行性，完事之后会吧它改成速度快一点的静态语言，这里用的是 c #,我就是做的c # 算法端的东西，前端可视化数据是另外一位lead researcher 在做。走之前把所有接口实现了，实现了个简单的console 的demo。前端那位和我同名的 researcher 就开始干活了。

工程上的感受

其实我来之前很多东西都不会，比如git, c # ,我一直觉得我们所学的东西可以分为3种层次

idom > knowledge > tool

这些东西一个比一个学起来简单，语言语法和git就是完全属于tool的一类东西，而一些理念就是属于idom一类的东西，比如: 软件工程上的设计理念，functional,oop,架构设计，以及一些更加属于哲学 范畴上的东西，那些东西很难学，但是一旦学会就很少会忘记，并且学习成本不会太高，而knowledge 更加属于像原材料一样的东西，有了这些东西你进一步的做出新东西，比如我们本科学的那些东西，没有这些东西呢，基本这些公司就不大会要你。其实tool则根本不用学，应为基本上在工作中学，或者直接上网查会更快。我就是这样，当初学git的时候，因为就是一个人在开发根本用不到branch的概念所以基本学了那一部分都不会有用，因为tool你不用就忘记了。事实证明我是对的。git的那些技能就俩命令 ：pull request, pull,merge

什么 廖雪峰之类的大段长文根本没用，tool 就是你不用就会忘记的东西，所以惰性求值就好了

c# 也是，我来之前根本没有学过，用都没用过，直接看了一份stackoverflow 的keyword 比对就开始用了，我感觉语言之间的迁徙并没有太大的壁垒因为当前主流的程序语言总是两大范式 : FP,OOP 这两基本的东西弄明白了，基本就是一个多语言开发者了，所以说科班基础还是很重要的。

研究院的感受

这边大牛确实多，我的mentor 是清华基科班毕业的，清华的硕士，这边好像很多都是 thu,pku的。我感觉那层楼都是名校毕业的，实习生也大多thu,pku,科大，北航…个人有点格格不入，开始的时候mentor每次介绍我是jnu的本科生我还有一点害羞，但后来就渐渐好了，因为这确实是不争的事实，仅仅看名校学历和之前看门庭有什么区别呢。后来我就渐渐的放下了，1来这是事实，我不争，2来我觉得，有一些地方确实比别人差，我要承认，3来我要对自己的能力有清晰的认识，差就是差，行就是行。这是和学校无关的。和自己的修行有关。
